Combine objects while summing same keys.

In-place modification of the first object passed to the function.

## Installation

`$ npm install --save sum-object-keys`

## Usage

```js
import sum from 'sum-object-keys';

const result = sum(a,b);
// result is ...
```

```
sum(target, ...sources);
```
